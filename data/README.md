## 4. aCM Results (from Clomics pipeline)

The chromatin module (CM) inference was performed externally using the **Clomics** pipeline on the SEA-AD snATAC-seq data. The outputs used in these notebooks are:

- `count_matrix.txt` — Pseudobulk aCM score matrix (CMs × donors)
- `metadata.txt` — Donor-level metadata aligned with Clomics output
- `results_aCM_AD/` — Correlation results between aCM scores and AD pathology

These intermediate files are generated by the Clomics pipeline and are tracked in this repository (small enough to include).
